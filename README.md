# Descrição
## Esse projeto é uma simples to-do list feita utilizando *ReactJS*, *Redux* e *TailwindCSS*
- Projeto totalmente responsivo
- Animações ao clicar nas checkboxes e ao passar o mouse sobre o botão
- Tema claro e escuro baseado no tema do dispositivo
